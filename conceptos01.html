GIT Es un sistema de control de versiones que nos ayuda a gestionar nuestros archivos de proyecto.


GITHUB Es un servicio basado en la web para el control de versiones usando Git. Basicamente, es un sitio para desarrolladores, mediante codigos se identifican problemas y se propones cambios.


DIFERENCIAS ENTRE GIT Y GITHUB


GIT: Es una herramienta de control de versiones distribuidas que se pueden gestionar, el historial de codigos de fuente de un proyecto en desarrollo.


GITHUB: Es un servicio basado en la web para el control de versiones usando Git. Basicamente, es un sitio para desarrolladores, mediante codigos se identifican problemas y se propones cambios.
