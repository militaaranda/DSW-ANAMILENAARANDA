    GIT Es una herramienta gratuita y de codigo abierto para el control de versiones en forma distribuida, diseñado para manejar desde un proyecto pequeño hasta un gran proyecto en forma rapida y eficiente. GITHUB Es un repositorio web basado en Git que presta servicios de hosting, ofrece todas las opciones de control de versiones distribuido y la administracion del codigo fuente similar a Git, asi mismo agrego algunas funciones adicionales.
